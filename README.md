# Akshat Kumar Portfolio Hub

Welcome to the central hub for Akshat Kumar's professional portfolios. This repository contains a sleek, modern landing page that connects multiple tailored portfolio experiences.

## 🚀 Live Hub

You can access the live portfolio hub here: [akshat.shtar.space](https://akshat.shtar.space)

## 📁 Repository Structure

- **`/` (Root)**: Contains the main Landing Page hub. It uses a premium dark-themed design with smooth micro-animations.
- **`/akshat-pro`**: The Pro-tier portfolio experience.
- **`/akshat-ultra`**: The Ultra-tier portfolio experience (Framer-based).
- **`/akshat-extreme`**: (Coming soon) The Extreme-tier portfolio experience.

## 🛠️ Technologies Used

- **HTML5 & CSS3**: Custom vanilla CSS for maximum control and performance.
- **JavaScript**: Native ES modules and React (for the Ultra section).
- **Glassmorphism & CSS Gradients**: Used extensively to create a premium, high-contrast visual experience.
- **GitHub Pages**: Automated deployment via GitHub Actions.

## 💻 Local Development

To run this project locally, simply start a local web server at the root directory:

```bash
python3 -m http.server 3000
```

Then navigate to `http://localhost:3000` in your web browser.

## 🎨 Design System

- **Primary Colors**: Deep Black (`#050505`), Pure White (`#ffffff`), Subtle Grey (`#888888`)
- **Typography**: Inter (Google Fonts)
- **Effects**: Radial background glows, blurred backdrop filters, and subtle translation transforms on hover.
